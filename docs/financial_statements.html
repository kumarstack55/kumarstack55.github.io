決算書¶

企業が公開する書類¶

  • 上場企業

    • 有価証券報告書(年1回)

    • 四半期報告書(3カ月に1回)

  • 非上場企業

決算書の中身¶

  • 財務3表

    • 貸借対照表(=B/S)

    • 損益計算書(=P/L)

    • キャッシュフロー計算書(=C/S)

  • 株主資本など変動計算書(=S/S)

決算書の想定読者¶

  • 例: 企業内部

    • 現金、売掛金、借入金がどの程度かなど、今後の打ち手を考える上で考慮すべき重要な事項だから。

  • 例: 銀行・投資家

    • 利息、配当金、株価の値上がり益などのリターンを見込む検討材料になるから。

  • 例: 企業間取引

    • 財政、経営成績から企業の安定性を分析するため。

貸借対照表(B/S)¶

  • 左側: 借方, 右側: 貸方

  • 左側: 資産の部 (運用状況)

    • 流動資産 ≒だいたい1年いないに資金回収できるもの。

      • 例: 現金、銀行の預金、受取手形、製品など

    • 固定資産

  • 右側: 負債の部 (調達状況)

    • 負債

      • 流動負債

        • 例: 支払い手形、買掛金、短期借入金など

      • 固定負債

        • 例: 長期借入金、社債、退職給付引当金など

    • 純資産 ≒返済が不要な資金

      • 株主資本(株主に帰属する)

        • 株主資本は、その企業が真に自由に使うことができるお金

      • その他(含み益など)

債務超過¶

資産 = 負債 + 純資産 であるため、 もし 資産 < 負債 のとき、純資産はマイナスとなる。

この状態を債務超過という。

原価率¶

原価率(%) = 売上原価 / 売上高 * 100
