## Markdown

### プレビューを開く。

`Ctrl + Shift + V`

## 拡張機能

### Paste Image

```
名前: Paste Image
ID: mushan.vscode-paste-image
説明: paste image from clipboard directly
バージョン: 1.0.4
パブリッシャー: mushan
VS Marketplace リンク: https://marketplace.visualstudio.com/items?itemName=mushan.vscode-paste-image
```

クリップボードの画像イメージを Markdown などにペーストできる。

Windows の場合、 `Ctrl + Alt + V` でカーソル位置にペーストする。

### Markdown PDF

```
名前: Markdown PDF
ID: yzane.markdown-pdf
説明: Convert Markdown to PDF
バージョン: 1.4.4
パブリッシャー: yzane
VS Marketplace リンク: https://marketplace.visualstudio.com/items?itemName=yzane.markdown-pdf
```

Markdown ファイルを印刷しやすい pdf にする。

Markdown のタブ内で右クリックし、コンテキストメニューで `Markdown to PDF: Export (pdf)` を選ぶ。

### Vim

```
名前: Vim
ID: vscodevim.vim
説明: Vim emulation for Visual Studio Code
バージョン: 1.23.2
パブリッシャー: vscodevim
VS Marketplace リンク: https://marketplace.visualstudio.com/items?itemName=vscodevim.vim
```

Vim のキーバインディングで操作する。
